# 인공지능 프로젝트 기획서

## 1. 프로젝트 개요

### 1.1 프로젝트명
NotebookLM 클론 - 문서 기반 AI 어시스턴트

### 1.2 프로젝트 목표
Google NotebookLM과 유사한 문서 기반 AI 어시스턴트를 구현합니다.
사용자가 업로드한 문서를 기반으로 질문에 답변하고, 요약, 퀴즈 등을 생성합니다.

### 1.3 핵심 기능
1. 문서 업로드 및 관리
2. 문서 기반 질의응답 (RAG)
3. 자동 요약 생성
4. 플래시카드 및 퀴즈 생성
5. 출처 인용 기능

## 2. 기술 스택

### 2.1 백엔드
- Python 3.11+
- FastAPI 프레임워크
- LangGraph 워크플로우 엔진

### 2.2 AI/ML
- Gemini File Search API
- Gemini 2.5 Flash 모델

### 2.3 인프라
- Docker 컨테이너화
- PostgreSQL 데이터베이스
